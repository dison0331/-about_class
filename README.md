# 教务系统调课软件（高中版）

一个基于Python开发的高中教务系统调课管理工具，支持课表显示、调课操作和冲突自动检测。

## 功能特点

- ✅ **课表管理**: 添加、删除、修改课程信息
- ✅ **课表显示**: 按天、按老师查看课表
- ✅ **智能调课**: 支持调整课程的星期和节次
- ✅ **冲突检测**: 自动检测老师时间冲突并提示
- ✅ **数据持久化**: 支持课表保存和加载（JSON格式）
- ✅ **友好界面**: 命令行交互界面，操作简单直观
- 🎓 **高中课程**: 专为高中课程设计，包含语数英、理化生、史地政等科目

## 系统架构

### 核心类

#### Course（课程类）
- 存储课程基本信息：课程ID、名称、老师、时间等
- 提供课程数据序列化/反序列化功能

#### ScheduleManager（课表管理器）
- 管理所有课程数据
- 实现冲突检测算法
- 提供课表查询和显示功能
- 支持数据持久化

## 使用指南

### 主菜单功能

1. **添加课程**
   - 输入课程ID、名称、老师姓名
   - 设置上课时间（星期几、开始节次、持续节次）
   - 系统自动检测冲突

2. **显示课表**
   - 查看完整课表
   - 查看特定星期的课表
   - 查看特定老师的课表

3. **调课功能**
   - 选择要调整的课程
   - 调整星期、节次或两者都调整
   - 自动检测冲突并提示

4. **删除课程**
   - 根据课程ID删除课程

5. **保存课表**
   - 将当前课表保存到JSON文件

6. **加载课表**
   - 从JSON文件加载课表数据

7. **查看老师课表**
   - 按老师姓名查看其所有课程

## 冲突检测机制

系统会自动检测以下冲突：
- **老师时间冲突**: 同一老师在相同时间有多个课程
- **冲突提示**: 显示具体的冲突信息，包括冲突的课程名称和时间

## 数据结构

### 课程数据格式
```json
{
  "course_id": "CHN101",
  "name": "语文",
  "teacher": "张老师",
  "day": 1,
  "period": 1,
  "duration": 2
}
```

### 字段说明
- `course_id`: 课程唯一标识符
- `name`: 课程名称（高中科目名称）
- `teacher`: 授课老师姓名
- `day`: 上课星期（1-7，1表示周一）
- `period`: 开始节次（1-12）
- `duration`: 课程持续节次

## 高中课程特点

### 课程安排特点
- 每节课通常为45分钟
- 主科（语数英）通常每天或隔天安排
- 理科/文科科目根据学生选科情况安排
- 体育、美术、音乐等课程每周1-2次
- 实验课（理化生实验）通常安排连续2-4节

### 调课注意事项
- 避免同一老师同一时间冲突
- 考虑课程的前后衔接
- 注意实验室等特殊教室的占用情况
- 避免连续安排同一科目过多课时
